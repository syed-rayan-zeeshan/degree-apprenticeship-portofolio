##  Overview
This project allows users to input a category (e.g., "Sports", "Music", "News") and get personalized BBC programme recommendations.  
It demonstrates **Python fundamentals, data handling, and algorithmic logic**, all directly relevant for a Software Engineer apprenticeship.

---

##  Features
- Keyword-based content recommendations  
- Uses a JSON dataset for easy expansion  
- Command-line interface (CLI) for user interaction  
- Lightweight and beginner-friendly  

---

##  Skills Demonstrated
- Python 3 programming (loops, conditionals, functions)  
- JSON file handling and data structures  
- Algorithmic thinking for filtering and recommendation logic  
- User input processing
