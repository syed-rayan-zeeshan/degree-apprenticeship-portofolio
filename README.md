# degree-apprenticeship-portofolio
A beginner-friendly Python portfolio showcasing 10 projects, including BBC-themed data projects and general programming exercises. Each project includes working code, screenshots, and README documentation.


Featured BBC specific Projects (3)


BBC Sentiment Dashboard	Analyzes BBC headlines and classifies them as positive, neutral, or negative using keyword matching.	

BBC Audience Trend Analyzer	Reads BBC audience data and summarizes category popularity with a text-based bar chart.	

BBC Recommender	Suggests BBC programs based on user-selected categories from JSON data.	

General Beginner Python Projects (6)

Project	Description

Calculator	Basic arithmetic operations with user input.

To-Do App	Task manager for adding, viewing, and deleting tasks.

Number Guessing Game	Simple game where the user guesses a random number.

Password Generator	Creates random passwords of user-specified length.

Quiz App	Multiple-choice quiz with scoring.

Data Analysis	Basic data analysis program demonstrating loops and calculations.

Simple Budget Tracker	Tracks income, expenses, balance, and category totals. Stores data in JSON so it persists between sessions.
