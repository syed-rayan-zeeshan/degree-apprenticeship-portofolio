# Simple Budget Tracker (Beginner Version) 

A beginner-friendly Python project that helps users track income, expenses, balance, and category totals. All data is stored in a JSON file so it persists between sessions.



## Overview
This project demonstrates:
- Basic Python programming (loops, conditionals, variables)
- File handling using JSON
- Simple arithmetic calculations
- Text-based menu for user interaction

Users can:
1. Add income
2. Add expenses
3. View current balance
4. View total expenses by category
5. Exit the program



## Features
- Beginner-friendly Python code  
- Persistent data storage with JSON  
- Simple and clear text-based interface  
- Easy to expand with more features
