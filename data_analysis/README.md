# BBC Viewership Data Analysis



A beginner-friendly Python project that reads a CSV file of BBC show viewership data and provides simple analysis.



## Features

- Calculates total viewers across all shows

- Finds the most-watched and least-watched shows

- Demonstrates reading CSV files and basic Python calculations
