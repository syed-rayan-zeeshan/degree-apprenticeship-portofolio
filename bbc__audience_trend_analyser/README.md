## Overview
This project reads audience data from a JSON file and calculates:
- Total views for each BBC category (News, Sport, Music, etc.)
- Which category is most popular overall
- A simple text-based bar chart showing popularity

---

## Features
- Uses only core Python (no external libraries)
- Reads and processes JSON data
- Generates a text-based bar chart
- Displays totals clearly and neatly

---

## Skills Demonstrated
- Python file handling
- Working with JSON data
- Loops and conditionals
- Sorting and basic data visualization (with text bars)
